Important FAQ


1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree-like hierarchy, where each element is a node. The DOM allows JavaScript to access, modify, and manipulate the content, structure, and styles of a webpage dynamically.

2. How do you select an element from the DOM?

To select an element from the DOM, i can use different methods based on my needs. If i want to select an element by its ID, getElementById(), which returns a single element. To select elements by class name, use getElementsByClassName(), which returns a collection of matching elements. If I need to select elements by tag name, I can use getElementsByTagName(), which also returns multiple elements. For more flexibility, CSS selectors can be used with querySelector() to get the first matching element or querySelectorAll() to get all matching elements. Among these, querySelector() and querySelectorAll() are the most powerful and widely used methods.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where a parent element handles events for its child elements, even if they are added dynamically. Instead of attaching event listeners to multiple child elements, you attach a single listener to their common parent and use event bubbling to detect which child triggered the event.

4. How do you manipulate an element's attributes and styles using the DOM?

In the DOM, you can manipulate an element’s attributes and styles using JavaScript. If you want to change an attribute of an element, you can use setAttribute() to modify it, getAttribute() to retrieve its value, and removeAttribute() to delete it. For example, if you have an image and want to change its source dynamically, you can use setAttribute("src", "new-image.jpg").
Similarly, you can modify an element’s styles using the style property. For example, element.style.color = "blue" changes the text color to blue. You can also apply multiple styles at once using style.cssText. However, instead of directly modifying styles, a better approach is to use classList, which allows you to add, remove, or toggle CSS classes efficiently. For example, element.classList.add("new-class") applies a predefined CSS style, making your code cleaner and easier to manage.
